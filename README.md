# Address-Bloc
My first Ruby application is an address book.
 
## Features:
* Ruby program from the command line
* Interactive menus
* Ruby models used to represent data
* Import entries from a file
* Search
* Testing
 
## Technology Focus:
Ruby, Command line, Cloud-9
